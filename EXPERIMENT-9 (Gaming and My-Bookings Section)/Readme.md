# Project Module: Gaming & My Bookings

## Overview
Module 5 introduces a **mini-game interface** and a **personal bookings page** for users. It enhances engagement while providing easy management of movie bookings.

## Features

1. **Mini-Games**
   - Users can play interactive games while waiting for showtimes.
   - Games are lightweight and browser-friendly.
   - Scores and rewards can be linked to user accounts (optional).

2. **My Bookings Page**
   - Displays a list of all bookings made by the user.
   - Users can view booking details: movie, showtime, seats, and total amount.
   - Options to cancel or reschedule bookings if allowed.

3. **Booking Summary**
   - Provides a quick summary of upcoming and past bookings.
   - Shows ticket download links and booking status.

4. **Real-Time Updates**
   - Any changes in booking status are updated in real-time.
   - Ensures data consistency with backend systems.
