# Project Module: Admin Panel

## Overview
Module 4 handles the **administrative management** of the CineNext movie booking system. It provides a web-based interface for admins to efficiently manage movies, shows, users, and bookings.

## Features

1. **Dashboard**
   - Provides an overview of total movies, shows, users, and bookings.
   - Displays analytics like popular movies, upcoming shows, and booking trends.

2. **Movie & Show Management**
   - Add, edit, or remove movies and their showtimes.
   - Ensure shows are properly linked to theaters and timings.

3. **User Management**
   - View all registered users.
   - Block or unblock users if necessary.
   - Track user activity and booking history.

4. **Booking Management**
   - View all bookings in real-time.
   - Filter bookings by date, show, or user.
   - Cancel or modify bookings if required.

5. **Reports & Analytics**
   - Generate reports on sales, bookings, and revenue.
   - Export data for external analysis.
